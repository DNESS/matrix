# Matrix

The matrix for your terminal.

![Matrix](https://github.com/prggmr/matrix/raw/master/image.png)

## Requirements

* XPSPL

## Running

```sh

git clone ~/git@github.com:prggmr/matrix.git .matrix
xpspl ~/.matrix/matrix
```

Install to bin.

```sh

sudo ln -s $HOME/.matrix/matrix /usr/bin/
```

Then run

```sh
matrix
```
